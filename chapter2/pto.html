PTO Program

Version 1

A simple program that calculates the amount of PTO for 1 year for an employee that receives 5 hours per paycheck and receives a paycheck bi-weekly

  1. Display 5 hours per paycheck
  2. Calculate how many paychecks received for one year if paid bi-weekly
  3. Display PTO for 1 yr

Start

SET INT hours = 5

SET INT payCheck = 26

DISPLAY "This is a simple program that outputs how many hours you earn a year for PTO."

SET INT PTO = hours * payCheck

DISPLAY "You earn hours per year."

End


Version 2

  1. 1-5 yrs = 5 hours
  2. 5 yrs = 6 hrs
  3. 10 yrs = 7 hrs
  4. PTO for 1 yr in weeks

Start

Get years

Set payCheck = 26

If years < 5, PTO = 5 * payCheck

If years equal to or > 5, PTO = 6 * payCheck

If years equal to or > 10, PTO = 7 * payCheck

Set PTOWeek = PTO / 40

Display PTOWeek

End
