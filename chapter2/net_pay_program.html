Net Pay Calculator

Calculate the net pay for an employee.It takes the gross pay and deducts Federal, State, FICA. It has the option to deduct any additional amount from the gross amount. Then it displays the net pay.

  1. START
  2. SET grossPay
  3. DISPLAY Enter your gross pay
  4. INPUT grossPay
  5. Set federal = 0.15
  6. Set state = 0.05
  7. Set FICA = .0762
  8. Set net = grossPay - ((grossPay * federal) + (grossPay * state) + (grossPay * FICA))
  9. DISPLAY Any additional deductions?
  10. SET deduct
  11. INPUT deduct
  12. SET netPay = net - deduct
  13. DISPLAY netPay
  14. END
