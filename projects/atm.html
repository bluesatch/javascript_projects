Logic

  • DISPLAY "Enter your card"
  • INPUT card
  • DISPLAY Enter your PIN code
  • INPUT pin
  • DISPLAY "What would you like to do?"
    • Withdraw
    • Deposit
    • Check Balance
    • See Bank Statement
  • If Withdraw:
    • DISPLAY "How much would you like to withdraw?
    • INPUT amount
    • IF balance is ok
    • DISPLAY "Dispensing funds. Have a nice day."
    • ELSE
    • DISPLAY "Insufficient funds on this card
  • ELSE IF Deposit
    • DISPLAY "How much would you like to deposit?"
    • INPUT depositAmount
    • ADD depositAmount and balance
    • DISPLAY "Here is your new balance
  • ELSE IF Check Balance
    • DISPLAY balance
  • ELSE IF See Bank Statement
    • DISPLAY "Sorry our system is down. Please check balance on our website or inside."
