Word Counter

Take the input from the user and return the number of cosonants and vowels in the input.

  1. START
  2. SET vowels = [a, e, i, o, u]
  3. SET consonants = [b, c, d, f, g, h, j, k, l, m, n, p, q, r, s, t, v, w, x, y, z]
  4. SET inp
  5. DISPLAY "Enter a sentence"
  6. INPUT inp
  7. Count the characters in inp
  8. For characters in inp DISPLAY total
